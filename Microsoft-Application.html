برامج مايكروسوفت التعليمية

Microsoft Visio

يستخدم لإنشاء مخططات ورسومات توضيحية للمفاهيم التعليمية المعقدة

Microsoft Power BI

يُستخدم لتحليل البيانات وإنشاء تقارير مفصلة لدعم قرارات التعلم والتدريس

Microsoft Forms

يتيح إنشاء استطلاعات واختبارات عبر الإنترنت لتقييم فهم الطلاب والحصول على ردود فعل فورية

Microsoft PowerPoint Presenter Coach

يوفر تقديم ملاحظات وتوجيهات لتحسين مهارات التقديم والتواصل لدى الطلاب

Microsoft Math Solver

يوفر حلولًا لمشاكل الرياضيات والعمليات الحسابية بواسطة الصورة أو الكتابة اليدوية

Microsoft Whiteboard

يمكن استخدامه للتعاون على الرسومات والمخططات التوضيحية بين الطلاب والمعلمين

Microsoft Education Center

يقدم موارد تعليمية ودورات تدريبية للمعلمين والمدرسين لتعزيز مهاراتهم التعليمية

Microsoft Sway

يساعد في إنشاء تقارير وعروض تقديمية تفاعلية وجذابة لتقديم المواد التعليمية

Microsoft Minecraft

Education Edition: يوفر بيئة تعليمية افتراضية تستند إلى لعبة Minecraft لتعزيز التعلم والتفاعل.

Microsoft Translator

يتيح ترجمة النصوص والمحادثات في الوقت الفعلي لدعم الطلاب من مختلف الثقافات واللغات.
